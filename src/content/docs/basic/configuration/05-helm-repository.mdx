This page allows you to configure the Helm repositories used by the OtterScale system. Helm repositories are collections of charts that describe how to install and manage Kubernetes applications.

## Overview

The Helm Repository settings page displays the currently configured Helm repository URLs. Each repository is represented by a package icon.

## Edit Helm Repositories

To manage the list of Helm repositories, follow these steps:

<Steps>

1.  Click the **Edit** button (pencil icon) located at the top right of the configuration section.

2.  A modal window titled "Edit Helm Repository" will appear.

3.  In the **URL** field, you can manage the list of repository URLs:

    - **Add a repository:** Type the Helm repository URL (e.g., `https://charts.bitnami.com/bitnami`) in the input field and click the **Add** button (or press Enter).
    - **Remove a repository:** Click the remove icon next to an existing URL to delete it.
    - **Clear all:** Use the **Clear** button to remove all entries.

4.  Click **Confirm** to save your changes. The system will update the configuration and the new list will be displayed.

</Steps>
